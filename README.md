# Smart-Tourist-Safety-Monitoring-Incident-Response-System
=======
# WanderMate 🛡️

**Smart Tourist Safety Monitoring & Incident Response System**

Built for Smart India Hackathon 2025 (SIH25002) by Team WanderBytes.

## Features
- SOS Panic Button with real-time alerts
- Live GPS tracking with Leaflet maps
- Geo-fencing & AI anomaly detection
- Blockchain-based Digital Tourist ID
- Offline support with auto-sync
- Admin dashboard for authorities

## Tech Stack
- **Frontend:** React 18 + Tailwind CSS + Leaflet
- **Backend:** Node.js + Express + Socket.io
- **Database:** In-memory (swap to MongoDB for production)

## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm start
```

### Mobile APP
```bash
cd mobile
npm install
npx expo start
```

Backend runs on `http://localhost:5000`, Frontend on `http://localhost:3000`.
